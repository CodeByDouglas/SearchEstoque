Busca Sequencial

Percorre cada item do estoque um por um até encontrar o desejado. É o método mais simples, porém pode se tornar lento à medida que o número de produtos cresce.

Complexidade: O(n)

Busca Indexada

Utiliza índices otimizados do banco de dados (SQL) para localizar o item diretamente. Muito mais eficiente que a busca sequencial para grandes volumes de dados.

Complexidade: O(log n)

Busca HashMap

Carrega os dados em memória utilizando uma Tabela Hash. Permite acesso quase instantâneo aos produtos, oferecendo a melhor performance possível.

Complexidade: O(1)
